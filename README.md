# Eats and Flix

## Ready for fun?!

Plan your dinner and movie night out using this awesome app! By entering a zip-code and the date of your preferred night out, Eats and Flix will provide search results for restaurants and films within a user-specified radius - with driving directions provided by Google Maps. Furthemore, Eats and Flix affords users the options of viewing restaurant websites and menus, making reservations (if applicable), and connecting to Fandango to buy tickets online.

## Eats and Flix was built using:

* HTML5
* CSS3
* JavaScript and jQuery
* Ajax-JSON
* Node.js
* Express.js
* React and React-DOM
* Multiple APIs
